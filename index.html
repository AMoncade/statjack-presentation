Esperance Mathematique (EV)

Simulations de Monte Carlo — 1 000 iterations

Principe

Pour chaque simulation :
1. Sabot neuf, distribution de 2 cartes
2. Strategie de base simplifiee
3. Evaluation : BJ = +1.5 | Gain = +1 | Nul = 0 | Perte = -1
4. Moyenne cumulative ➔ courbe de convergence

EV = somme(gains) / N simulations
